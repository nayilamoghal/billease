# 💰 BillEase – Billing & Invoice System

A complete **Billing and Invoice Management System** built as a Final Year BCA Project.

## 🔗 Live Demo
> After deploying on GitHub Pages, your link will be:
> `https://YOUR-USERNAME.github.io/billease/`

---

## ✨ Features

- 📊 **Dashboard** – Total revenue, invoices, customers, pending amount with live charts
- 📄 **Invoice Management** – Create, view, delete invoices with GST calculation
- 👥 **Customer Management** – Add, edit, delete customers with full details
- 📦 **Products & Services** – 45 pre-loaded products across 6 categories
- ⚡ **Auto Price Fill** – Select a product in invoice and price fills automatically
- 🧾 **GST Calculator** – Supports 5%, 12%, 18%, 28% GST with CGST + SGST split
- 📥 **PDF Download** – Download professional invoice as PDF
- 📈 **Reports & Analytics** – Charts for revenue, top customers, GST breakdown
- 💾 **Local Storage** – All data saved in browser automatically

## 🚀 How to Run Locally

1. Download or clone this repository
2. Open `index.html` in any browser
3. No installation needed — works offline!

```bash
git clone https://github.com/YOUR-USERNAME/billease.git
cd billease
# Open index.html in browser
```

---

## 🌐 Deploy on GitHub Pages (Free Hosting)

1. Push code to GitHub
2. Go to your repo → **Settings** → **Pages**
3. Under **Source**, select `main` branch → `/ (root)`
4. Click **Save**
5. Your live link: `https://YOUR-USERNAME.github.io/billease/`
